I've been building for a while without writing much about it. This is me changing that.

The plan here is simple: short notes on what I'm working on, decisions I made and why, and the occasional thing I learned the hard way. Less polished than a tutorial, more honest than a changelog.

If a project taught me something about architecture, performance, or just shipping — it'll probably show up here.

```ts
export function ship(idea: Idea) {
  const built = buildFast(idea);   // learn what matters
  const refined = slowDown(built); // raise the bar before anyone sees it
  return refined;
}
```
